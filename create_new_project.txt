1. Створити package.json командою npm init (можна додати прамор -y який відповість yes на всі питання)
2. Встановити потрібні пакети npm install 'package name' 
3. Створити .gitignore і прописати папку node_modules
4. Прописати скрипти

5. Для автоматичного перезапуску встановити пакет nodemon -D 
6. Прописати скрипт 'dev' або 'start:dev': 'nodemon index.js'
7. Запустити npm run dev (всі скрипти крім start i test потрібно запускати через run).

module.exports = {user, contacts}; Використовуємо деструктуризацію щоб експортувати всі функції з поточного файлу
